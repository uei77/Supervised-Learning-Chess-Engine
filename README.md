# Reinforcement-Learning-Chess-Engine
Chess Engine is around 1900 2000 elo.
Chess Engine was developed using PyTorch
The engine utilizes Monte Carlo Tree Search (MCTS) with 800 simulations per move for optimal decision-making.
A FastAPI backend handles asynchronous move calculations and seamlessly integrates the model with the web interface.
Chess Engine has trained using 500000 games of Ajedrez 1600+ elo dataset and 360000 of lichess elite august 2020 dataset.
https://ajedrezdata.com/databases/otb/
https://database.nikonoel.fr/
Chess Engine played over 500 games on its own
Chess Engine played over 1000 games with Stockfish. Over 800 matches versus Stockfish 5, over 250 matches versus Stockfish 6.
Self-play evaluations and benchmarking were conducted against Stockfish 5 and 6.
https://stockfishchess.org/
